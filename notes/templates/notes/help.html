Shortcut list

There are many shortcuts available when editing documents. The current list of shortcuts is this:

Hyperlinks to other documents

A link to a document is created with the basic [[[]]] where the link is in the middle: example [[[my link]]]. This will only work if there exists a document with the name "my link", if not, "my link" will appear red in the text. If one wants the text to the link to be something else than the name of the document it is possible to create that like this: [[[name of my document | displayed text]]]

Links in the text

To create a list of links in the text (at the bottom of the document) on can do [links[[ description of the link 1 | actual link 1 ; description of link 2 | actual link 2 ]]]

"|" is used to separate the name of the link and the actual link, ";" is used to separate the links. There can be as many links as you like. They will appear as a list.

Escaped text

[esc[[text]]] is to print the text as is. This is mostly used in XML text where the parser gets confused with "<" and ">"

Java code block

[java[[text]]] displays the text as a block of Java code.

Images

[image[[img_name]]] displays images. One can choose which side it should float on like this: [image[[img_name | side=left]]] where side can be "right" or "left".

Using "width" it is possible to set how large the image is supposed to be. It needs to be a number and nothing more. example: "width=50"

Using "no_text" removes the text at the bottom of the image.

List documents

To list all documents connected to a tag write [list[[tag_name]]]. You can exclude specific documents with [list[[tag_name | excluded_doc_name_1, excluded_doc_name_2 ]]]
